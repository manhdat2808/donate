# Donate ALGO – Algorand Testnet dApp

A **unique decentralized application (dApp)** that allows users to send **ALGO on Algorand Testnet** to any wallet.

---

## Features

- Send ALGO to any Algorand address
- Real-time balance display
- Powered by **Pera Wallet** (mobile & browser)
- No gas fees – fully on **Testnet**
- Clean, responsive UI

---

## Live Demo

[Open dApp](https://manhdat2808.github.io/algorand-donate-dapp/)

## Setup & Usage

1. **Install Pera Wallet**  
   → [https://perawallet.app](https://perawallet.app)

2. **Get Testnet ALGO**  
   → [Faucet](https://testnet.algorand.network/faucet)

3. **Open dApp** → Connect wallet → Send ALGO!
